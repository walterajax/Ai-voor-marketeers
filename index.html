⚙️ Firebase instellen

Vul je Firebase-configuratie in firebase.js in voordat je de app gebruikt.

  1. Maak een project aan op console.firebase.google.com
  2. Activeer Realtime Database (kies testmodus)
  3. Ga naar Project Settings → Your apps → Web app
  4. Kopieer de firebaseConfig naar firebase.js
  5. Herlaad de pagina
